About

griddy is a low-level grid-based CSS framework, inspired by the awesome Tailwind CSS and originally built as a capstone project for The Odin Project's HTML and CSS course. It's written in Sass/SCSS.

Quick Start

  1. Copy griddy.css or griddy.min.css into your project directory.
  2. Link the CSS file in the head of your page. <link rel="stylesheet" href="dist/griddy.min.css">
  3. After you built your page, consider running the purge command to remove unused classes to reduce filesize greatly.

Containers

Containers in griddy are just div elements with a fixed size which depend on your breakpoints. The max-width of .container-sm is the same as the min-width set by the sm breakpoint, for example. By default, those are:

.container-sm 640px

.container-md 768px

.container-lg 1024px

.container-xl 1280px

.container always at 100%

Flexbox

With the .flex class, you can turn a container into a flexbox. You can then apply flex properties to it and its items.

You can control flex-wrap with the .wrap, .wrap-none or .wrap-reverse classes.

In addition to .flex and .flex-column, there are also the .flex-reverse and .flex-column-reverse classe, which flip the flex items around.

Grid

You can use the .order-{n} classes to customize the order of your grid items.
